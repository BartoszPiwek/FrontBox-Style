## Install

### Add package

```bash
yarn add @bpiwek/frontbox-style
# or
npm i @bpiwek/frontbox-style
```

### Import file

```scss
@import 'node_modules/@bpiwek/frontbox-style';
// or
@import '~@bpiwek/frontbox-style';
```
